# クリぽん

このアプリは「クリニックがぽんと見つかる」をコンセプトに、婦人科の病院を信頼できるレビューを元に探し易くすることを目的としています。

# 特徴
 
- 東京23区、女医の有無、診療科目で婦人科の病院を絞り込むことができます。
- ユーザーの領収書を使い、ユーザーが実際に病院を利用したことがあるかの承認を行います。
- 承認を行ったユーザーには承認済みを示すバッヂを表示します。
- 各病院にレビューを投稿することができます。
